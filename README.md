Medvis is a project developed to consider potential approaches to display medical records over time using React, TypeScript and chart.js.

https://user-images.githubusercontent.com/3259555/201749929-ff065a96-8f19-4d07-9d83-ec7bc5b22d1a.mp4

The instructions to run the project are the following, once you have installed yarn and expo:
- Run yarn install
- Then run expo start
- Select the web option from the Metro bundler by typing w

The prototype allows to select or remove patients and filter medical records according to atttributes values.
